# lobster-backup 🦞

> *"Your lobster has a memory. Don't let it drown."*

A backup and restore skill for [OpenClaw](https://openclaw.ai) agents. Frictionless install that just works, with optional encrypted remote storage.

## What It Does

- **Backs up** your lobster's workspace, memory, skills, config, cron jobs, and system-level customizations
- **Restores** everything to a fresh install — files and environment
- **Tracks** system changes via the Lobsterfile (structural memory, updated in real time)
- **Encrypts** backups with zero-knowledge key wrapping — only you can decrypt
